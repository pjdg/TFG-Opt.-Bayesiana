Para los nombres de los archivos excel y las gráficas se ha seguido la siguiente nomenclatura:

[Test Function][Kernel][Función adquisición]_[Parámetro trade-off]

Siendo la correspondencia numérica:

Test function:
	1 = Branin
	2 = Schwefel
	3 = Styblinski-Tang

Kernel:
	1 = RBF
	2 = Matern

Función adquisición:
	1 = PI
	2 = EI
	3 = UCB

Parámetro trade-off:
	001 = 0.01
	01 = 0.1
	02= 0.2

En el caso de las gráficas, se ha añadido una cifra adicional que indica la réplica a la que corresponde la gráfica:

[Test Function][Kernel][Función adquisición]_[Parámetro trade-off]_[Réplica]
